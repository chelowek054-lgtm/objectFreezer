Шаг 1: Модификация BlueprintCreator
Цель: сохранять в .blueprint полноценные текстовые эмбеддинги описания объекта.

После получения summary_text_prefixed (строка "<keyword>: <описание>") закодировать её тем же text_encoder (Qwen), который используется в ноде.

Не применять случайную матрицу R_sem.

Сохранить полученный тензор как ключ "blueprint_text_tokens" в safetensors.
Формат: [1, N_tokens, d_ctx], где N_tokens — длина последовательности, d_ctx — скрытая размерность (обычно 4096).

Опционально можно сохранить также средний эмбеддинг для обратной совместимости, но Injector должен использовать именно sequence.

Параметр: добавить поле num_text_tokens (INT, по умолчанию = 0, что означает «все токены»). Если >0, обрезать последовательность до первых N токенов.

Итог: старые ключи z_sem, z_hyb остаются для совместимости, новый ключ blueprint_text_tokens — основной для инъекции.

Шаг 2: Модификация BlueprintInjector
Цель: использовать blueprint_text_tokens вместо случайных проекций.

При загрузке .blueprint искать ключ "blueprint_text_tokens".
Если он есть:

Взять txt_tokens = blueprint_text_tokens[0] → форма [N, d_ctx].

Умножить на blueprint_scale (по желанию).

Вставить в txt‑контекст точно так же, как добавляются обычные текстовые токены (сейчас это делается в post_input).

Не вычислять W_cross @ z_hyb и не генерировать Blueprint‑токены из случайной проекции.

Количество добавляемых токенов = N_tokens, параметр K_cross игнорируется (или может быть использован для обрезки, если num_text_tokens было задано).

Параметр keyword_embedding по‑прежнему может вставляться (как отдельный токен имени), но теперь это опционально, т.к. описание уже содержит имя в начале.

Позиции (bp_tokens_position, keyword_position) работают как раньше.

Совместимость: если blueprint_text_tokens отсутствует (старый blueprint) — вывести предупреждение и пропустить инъекцию (или фоллбэк на старый метод, который сейчас не работает).

Шаг 3: Удалить/выключить случайные матрицы W_cross/W_self
Эти матрицы больше не нужны. Можно оставить код для экспериментов, но по умолчанию не использовать.
